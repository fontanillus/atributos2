Software para Desarrollo Web

Diseño Web Desarrollo Web Programación Avanzada de Sitios Web

El software de desarrollo web permite crear aplicaciones web desde sitios estáticos hasta aplicaciones dinámicas. Entre las herramientas más populares se encuentran:

Niveles de Desarrollo Web

Nivel I. Diseño de Webs Estáticas

  • HTML, CSS, JavaScript Básico: los pilares básicos para el desarrollo de páginas web.
  • Visual Studio Code (VS Code): editor de código abierto desarrollado por Microsoft.
  • GitHub: plataforma de código abierto para compartir y colaborar en proyectos.
  • Bootstrap: framework para CSS que ayuda a construir interfaces de usuario interactivas.
  • Chrome DevTools: herramienta para desarrolladores de navegadores.

Nivel II. Web reactivas y desarrollo de Webs Dinámicas

  • Servidor local: XAMPP, Laragon.
  • JavaScript Intermedio
  • Frameworks para JavaScript: React, Angular, Vue, Svelte.
  • Una de las siguientes bases de datos: MySQL, SQLite, Microsoft SQL Server.
  • Un lenguaje de BackEnd: PHP, Python, TypeScript y NodeJS.

Nivel III. Programación Avanzada de Webs: Frameworks, gestores de dependencia, MVC y POO

  • JavaScript Avanzado
  • POO en lenguaje de BackEnd: PHP, Python o JavaScript / NodeJS.
  • Frameworks para BackEnd: Laravel, Django, Express.js.
  • Gestores de dependencias: Composer, Miniconda, NPM.
  • Patrones de diseño: MVC.
  • ORM (Object Relational Mapping) para la base de datos seleccionada.
